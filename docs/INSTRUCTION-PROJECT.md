# Projeto Overview
O projeto é ERP para gestão empresarial dos seguintes segmentos:
    - Comércio
    - Indústria
    - Serviços 
    - Distribuição
Terá vários módulos:
    - Configuração
    - Contabilidade
    - Financeiro
    - Comercial
    - Vendas
    - Estoque
    - Suporte
    - Transporte
    - Produtos
    - Relatórios
    - Segurança
    - Gestão de Documentos
    - Gestão de Projetos
    - Gestão de Clinica Médica
    - Gestão de Consultórios
